# 区域数据可视化工具配置说明

## 概述

`config.py` 是一个统一的配置脚本，可以方便地管理和运行地图生成、热力图生成和分析报告生成功能。本工具支持任何城市或区域的数据可视化分析。

## 功能特性

### 1. 地图生成模式
支持3种区域名称显示模式：
- **all**: 显示所有区域名称
- **partial**: 只显示有数据的区域名称  
- **none**: 不显示任何区域名称

### 2. 热力图生成
- 生成客户流失率热力图
- 百分比格式显示，更直观

### 3. 分析报告生成 
- 自动生成详细的客户流失率分析报告
- 按标准模板格式输出统计数据
- 包含区域统计、年度趋势、排名分析等

### 4. 批量处理
- 可以一次性生成所有模式
- 支持完整套装（地图+热力图+分析报告）
- 自动组织文件结构

## 使用方法

### 方法1：交互式菜单运行
```bash
python config.py
```

然后按提示选择：
```
1. Generate map - Show all region names
2. Generate map - Show partial region names (data regions only)  
3. Generate map - Hide all region names
4. Generate heatmap
5. Generate customer churn analysis report
6. Generate all 3 map modes + heatmap
7. Generate all 3 map modes + heatmap + analysis report
8. Custom mode selection
0. Exit
```

### 方法2：直接调用函数
```python
import config

# 生成单一模式地图
config.run_map_generation(['partial'])

# 生成热力图
config.run_heatmap_generation()

# 生成分析报告
config.run_analysis_report_generation()

# 生成所有模式
config.run_map_generation(['all', 'partial', 'none'])
```

### 方法3：直接调用分析报告
```python
import heatmap
heatmap.generate_analysis_report()
```

### 方法4：演示脚本
```bash
python demo.py
```
一次性生成所有模式的地图和热力图

## 分析报告功能详解 

### 报告内容
分析报告严格按照标准模板格式生成，包含：
- **统计区域数量**：参与分析的区域总数
- **年份范围**：数据覆盖的年份范围
- **流失率范围**：最低和最高流失率（百分比格式）
- **最高流失地区**：流失数量最多的地区及详细信息
- **年度趋势分析**：各年份平均流失率统计
- **区域排名**：各区域平均流失率从高到低排序

### 报告模板格式
```
各区客户流失率热力图分析报告
============================================================

统计区域数量：X 个区
年份范围：XXXX - XXXX
流失率范围：XX% - XX%

最高流失数量地区：XXX区（XXXX年）
流失率：XX%
对应流失数量：XXX 人

各年份平均流失率：
  2022年：X.XX%
  2023年：X.XX%
  2024年：X.XX%
  2025年：X.XX%

各区域平均流失率（从高到低）：
 XX区：XX%
 XX区：XX%
 XX区：XX%
 XX区：XX%
============================================================
```

### 输出文件
- **文件名**：`流失率分析报告.txt`
- **编码**：UTF-8
- **位置**：项目根目录

## 数据文件要求

使用本工具需要准备以下数据文件：
- `heat_map.csv` - 客户流失数据（必需）
- `map.geojson` - 地理边界数据（必需）
- `shrink_ratio.csv` - 比例数据（必需）
- `customer_num.csv` - 客户数量数据（必需）

1. **编码错误**: 脚本已优化编码兼容性
2. **文件缺失**: 检查数据文件是否存在
3. **权限问题**: 确保输出目录有写入权限
4. **分析报告显示问题**: 在某些控制台环境中可能存在中文显示问题，但不影响文件生成
